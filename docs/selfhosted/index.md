# Self-Hosted

Run Streamient on infrastructure you control. The self-hosted edition is fully open source and includes the same product features as [Streamient Cloud](/cloud/).

## System Requirements

- **Node.js** >= 24
- **Docker** & Docker Compose (recommended)
- **pnpm** 10+
- **MongoDB** 6+
- **Redis** 6+
- **Typesense** 31+
