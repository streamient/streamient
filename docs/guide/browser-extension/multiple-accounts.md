# Multiple Accounts

You can add more than one Streamient account to the extension and switch between them without signing out. Each account keeps its own default project, URL/email routing, email connector, and automatic‑capture settings.

## Add an account

1. Click the **Streamient** icon in the toolbar.
2. Open the account menu.
3. Select **Add Account**.
4. Sign in with a different Streamient account.

## Switch accounts

Use the account menu to switch between accounts. The active account determines where new URLs, notes, and emails are saved from the popup and side panel.

## Remove an account

Open the account menu, select the account you want to remove, and choose **Remove Account**. Removing an account from the extension does not affect the account itself — it just signs you out of it locally.

## Per‑account settings

These settings live per account, so they don't bleed across:

- Default Project, URL Project, Email Project
- [Email Account connector](/guide/browser-extension/email-accounts) (IMAP/JMAP credentials)
- [Automatic URL capture](/guide/browser-extension/url-capture#automatic-capture) toggle, delay, scroll trigger, and exclude list

## Automatic capture across multiple accounts

If you enable [automatic URL capture](/guide/browser-extension/url-capture#automatic-capture) on more than one account, each captured URL is saved into every account that doesn't exclude it. Use the **Exclude sites** list per account to keep work and personal browsing in their own lanes.
