# Browser Extension

The Kumbukum Chrome extension lets you capture URLs, write notes, and work with email straight from your browser — without ever opening the Kumbukum web app. It auto‑detects email in Gmail, Outlook, and Fastmail, optionally syncs your mailbox over IMAP/JMAP, and can quietly capture pages you spend real time on so your bookmarks build themselves.

## What's in this section

- [Setup](/guide/browser-extension/setup) — install and sign in.
- [URL Capture](/guide/browser-extension/url-capture) — manual save from the popup and automatic capture based on time on page.
- [Email Capture](/guide/browser-extension/email-capture) — the email side panel: Summarize, Suggest reply, Internal notes, Show related, Email AI chat, one‑click ingest.
- [Email Accounts](/guide/browser-extension/email-accounts) — optional IMAP/JMAP connectors for Fastmail, Gmail, Outlook, and generic providers.
- [Multiple Accounts](/guide/browser-extension/multiple-accounts) — add and switch between multiple Kumbukum accounts.

## Where to start

- If you just installed the extension, do [Setup](/guide/browser-extension/setup).
- If you live in your email client, jump to [Email Capture](/guide/browser-extension/email-capture).
- If you want bookmarks to build themselves, see [URL Capture › Automatic capture](/guide/browser-extension/url-capture#automatic-capture).
